# 🇹🇬 TogoMarket - Le Marché Numérique du Togo

> Plateforme e-commerce locale pour acheter et vendre en toute confiance au Togo.

![TogoMarket Preview](https://via.placeholder.com/800x400/ea580c/ffffff?text=TogoMarket+Preview)

## ✨ Fonctionnalités

- 🛒 Catalogue de produits avec filtres et recherche
- 🔐 Authentification utilisateur (Client / Vendeur / Admin)
- 🛍️ Panier et commande en ligne
- 📦 Suivi de livraison
- 🏪 Gestion de boutique pour les vendeurs
- 📱 Design responsive (mobile-first)
- 🎨 UI moderne avec Tailwind CSS

## 🚀 Démo en ligne

👉 [Voir la démo](https://ton-username.github.io/togomarket) *(à mettre à jour après déploiement)*

## 🛠️ Technologies utilisées

- HTML5 / CSS3 / JavaScript (Vanilla)
- [Tailwind CSS](https://tailwindcss.com) via CDN
- [RemixIcon](https://remixicon.com) pour les icônes
- Stockage local : `localStorage` (aucun backend requis pour la démo)
